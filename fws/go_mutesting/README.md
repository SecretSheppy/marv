# go-mutesting

[go-mutesting](https://github.com/zimmski/go-mutesting) is a mutation testing framework for Go. It has a few maintained
forks that vary in operator level, but all output results in the same format. This makes all
of them compatible with Marv. go-mutesting is supported by Marv versions `1.2.1+`.

## Getting Started With go-mutesting In Marv

1. To get started with go-mutesting in Marv, run the `marv init` command to generate the required `.marv.yml`
configuration file.

```terminaloutput
marv init -f go-mutesting
```

2. Edit the field under the `go-mutesting` section in the `.marv.yml` file to point at the generated `report.json` file.
An example is shown below:

```yaml
# Enable the go-mutesting framework
go-mutesting:
    # The path to the report.json file generated by go-mutesting
    json-report: report.json
```

3. Run the `marv` command to launch Marv and click the localhost URL to open the Marv interface.

```terminaloutput
marv
```
